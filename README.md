# django-template
Django Template


| Command               | Function                           |
| --------------------- | ---------------------------------- |
| `make init`           | Add New Django Project             |
| `make install`        | Install requirements.txt & Migrate |
| `make user`           | Add Super User                     |
| `make new app=my_app` | Add New Django App                 |
| `make`                | Run Server                         |
| `make test`           | Run Tests                          |
| `make shell`          | Run Shell                          |
